# Avatar セットアップ手順

1. `public/` に使用したいアバター画像を配置し、`.env.local` に `NEXT_PUBLIC_HOME_AVATAR=/ファイル名` を設定します。  
   例: `NEXT_PUBLIC_HOME_AVATAR=/narufy_1024.png`
2. `.env` を変更したら **開発サーバを再起動** してください (`npm run dev` など)。
3. ブラウザで **Ctrl + F5**（ハードリロード）を実行し、キャッシュをクリアします。
4. `http://localhost:3000/avatar-test` を開いてプレビューし、`/ファイル名` を直接開いて HTTP 200 が返ることを確認します。
5. 問題がある場合は次をチェックしてください:
   - 先頭にスラッシュが付いているか？
   - 拡張子は正しいか？
   - `public/` 以下に存在しているか？
   - `.env.local` を修正後にサーバを再起動したか？
   - ブラウザキャッシュをクリアしたか？
